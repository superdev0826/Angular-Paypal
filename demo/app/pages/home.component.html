Usage

Configuration ( PayPalConfig)

Property Type Description
clientId string Id of your PayPal account
onApprove (data, actions) => void Executed when PayPal transaction was approved (but not authorized)
createOrderOnClient (data) => ICreateOrderRequest Use to create order on client. Either this, or 'createOrderOnServer' need to be defined
createOrderOnServer (data) => Promise<string> Use to create order on server. Either this, or 'createOrderOnClient' need to be defined
onClientAuthorization (authorization) => any Called when client authorization completes. Use this to notify your server about payment.
authorizeOnServer (data, actions) => any If set, you should authorize payment on your server. Default client authorization is not used in this case.
currency string PayPal's currency. Defaults to USD
button IPayPalButtonStyle Button configuration
onError (err) => void Called when PayPal experiences an error
onCancel (data, actions) => void Called when payment is cancelled
onClick () => void Called when user clicks on PayPal button
onShippingChange (data, actions) => any OnShippingChange callback as per https://developer.paypal.com/docs/checkout/integration-features/shipping-callback/
advanced IAdvancedConfiguration Advanced configuration
